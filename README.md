# proxychains-update

Automatically deletes proxychains ips and replaces them with new ones taken directly from www.spys.one

## dependencies

- python 3.9.x

- python modules ```requests, lxml, cssselect``` (```$ pip install requests==2.24.0 lxml==4.6.1 cssselect==1.1.0```)

- proxychains 4.x

## run

```$ ./start.sh```\
done
